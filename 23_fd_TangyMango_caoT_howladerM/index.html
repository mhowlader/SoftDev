Grid

Foundation has a 12 column grid that can scale out to any size. The grid can be utilized to create many different layouts. The size of the screen determines the proportion of the cell sizes for the grid.

Try resizing the screen to see how the gid changes.

These blue boxes are call outs.

8/8/8 cell.

2/4/6 cell.

Create a 6/8/10 cell 

        <div class="grid-x grid-padding-x">
          <div class="cell small-6 medium-8 large-10">
              <p>6/8/10 cell.</p>
          </div>
        </div>

Controls

Foundation controls include buttons, sliders(JavaScript), and switches.

Button Anchors: Primary Secondary Success Alert Warning
Button Actions: 
          <div class="grid-x">
            <div class="cell medium-6">
        <!-- Button Anchors: -->
                <a class="button" href="#">Primary</a>
                <a class="button success" href="#">Success</a>
                <a class="button alert" href="#">Alert</a>
            </div>
            <div class="cell medium-6">
        <!-- Button Actions: -->
                <button type="button" class="success button">Save</button>
                <button type="button" class="alert button">Delete</button>
                <br>
            </div>
          </div>

Button Sizes: tiny small Default large expanded small expanded 
            <div class="grid-x">
              <div class="cell medium-12">
                  Button Sizes:
    <!-- Replace large with one of the sizes above to change the size -->
                  <a class="button large" href="#">large</a>
              </div>
            </div>
Disabled Buttons: Disabled 
          <div class="grid-x">
            <div class="cell medium-12">
                <button type="button" class="button primary" disabled>Disabled</button>
                <button type="button" class="button secondary" disabled>Disabled</button>
            </div>
          </div>
